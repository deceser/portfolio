## Getting Started


```bash
git clone SSH/HTTPS
# next step
npm install
# next step
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) with your browser to see the result.

**Used technologies stack:**

- React/Vite
- React-router
- React-transition
- Bootstrap/Bootstrap-react

- Helmet-async
- Emailjs/browser
- .env
